# SQLiSense
SQLiSense is a lightweight, configuration-driven CLI tool that probes web endpoints for boolean- and time-based SQL injection vulnerabilities. Designed for developers, it produces clear, non-alarming reports and integrates easily into CI pipelines or local workflows.
